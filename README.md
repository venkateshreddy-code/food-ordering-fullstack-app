# 🍔 Food Ordering Full Stack Application

A **Full Stack Food Ordering Web Application** built using **Spring Boot, React, MySQL, Stripe, and AWS**.

The platform allows users to browse menu items, add items to a cart, place orders, complete secure payments, receive confirmation emails, and leave reviews.  
An admin dashboard allows administrators to manage categories, menu items, orders, and payment transactions.

# ✨ Features

### User Features
- Browse food categories
- Add items to cart
- Secure checkout with Stripe
- Email order confirmation
- Payment confirmation email
- Order history tracking
- Leave reviews for purchased items

### Admin Features
- Admin analytics dashboard
- Manage categories
- Manage menu items
- View and manage orders
- Track payment transactions
- Revenue statistics
